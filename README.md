# Build MerchFlow: A Headless Store API

### Project Description

You are going to build MerchFlow, a powerful product catalog API for an online clothing brand. You will set up categories, products, and different variations of those products like sizes and colors. But the real magic happens when you add the rules. You will write custom validation logic to make sure a t-shirt's price can never drop below zero, and you will learn how to make fields read-only so a regular shopper cannot magically change the inventory count of a sneaker. By the end, you will know how to protect your database with solid business logic built right into your serializers.

## Tech Stack

- Django REST Framework
